3、块元素水平垂直居中-通过position+margin

        .parent1 {
            width: 200px;
            height: 200px;
            position: relative;
            background-color: pink;
        }
        .child1 {
            width: 50px;
            height: 50px;
            position: absolute;
            background-color: green;
            top: 50%;
            left: 50%;
            margin-left: -25px;
            margin-top: -25px;
        }

4、块元素水平垂直居中-通过position+transform-不清楚子元素的宽高

        .parent2 {
            width: 200px;
            height: 200px;
            position: relative;
            background-color: pink;
        }
        .child2{
            width: 50px;
            height: 50px;
            position: absolute;
            background-color: green;
            top: 50%;
            left: 50%;
            transform: translate(-50%,-50%);
        }

5、块元素水平垂直居中-通过flex

        .parent3 {
            width: 200px;
            height: 200px;
            background-color: pink;
            display: flex;
            justify-content: center;
            align-items: center;
        }
        .child3 {
            width: 50px;
            height: 50px;
            background-color: green;
        }

5、块元素水平垂直居中-通过tab-cell

        .parent4 {
            width: 200px;
            height: 200px;
            background-color: pink;
            display: table-cell;
            vertical-align: middle;
        }
        .child4 {
